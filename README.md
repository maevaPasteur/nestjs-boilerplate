## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment

- [deployment documentation](https://docs.nestjs.com/deployment)
- cloud-based platform to deploy: [Mau](https://mau.nestjs.com)

```bash
$ npm install -g @nestjs/mau
$ mau deploy
```
